# 🌍 Multi-Agent AI Travel Planner

A Multi-Agent Travel Planning System built using LangGraph, Groq LLM, Tavily Search, OpenWeather API and Streamlit.

## 📂 GitHub Repository

## Features

- Destination Extraction Agent
- Flight Search Agent
- Hotel Recommendation Agent
- Weather Analysis Agent
- Budget Planning Agent
- Itinerary Generation Agent
- Final Report Generation Agent

## Tech Stack

- Python
- LangGraph
- LangChain
- Groq
- Tavily Search API
- OpenWeather API
- Streamlit
